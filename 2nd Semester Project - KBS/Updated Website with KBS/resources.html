Tools
The following are tools available to CampusNet Technicians for diagnosing and/or resolving hardware and software issues:
  • Hardware diagnostic tools:
    • Dell diagnostics via Dell boot menu
    • Drive Fitness Test, MemTest, TechTool Deluxe, etc via office CDs
    • PE USB
    • ASD/AST for Apple systems via office USB
    • Office internal HDD for PCs; install and boot
    • Office external HDD for Macs; connect via USB or FireWire and option-boot
    • Office network-connected NAS for Macs; connect via Ethernet and net-boot
  • Software diagnostic/resolution tools:
    • Data recovery tools: Y-Copy, GetBack, DataRescue, etc
    • OS installation tools: Windows auto-installers, Mac auto-installers
    • Malware removal tools: MalwareBytes, Spybot Search & Destroy, CCleaner, etc
    • Blue screen analysis tools: Verifier, WinDebug, etc
